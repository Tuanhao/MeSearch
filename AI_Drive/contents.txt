<p>Sometimes we want to remove all occurrences of a character from a string. There are two common ways to achieve this.</p>

<h2><span id="python-remove-character-from-string">Python Remove Character from String</span></h2>
<ol>
<li>Using string replace() function</li>
<li>Using string translate() function</li>
</ol>
<h3><span id="python-remove-character-from-string-using-replace">Python Remove Character from String using replace()</span></h3>
<p>We can use <a href="https://www.journaldev.com/23470/python-string-replace">string replace() function</a> to replace a character with a new character. If we provide an empty string as the second argument, then the character will get removed from the string.</p>
<p>Note that the string is <a href="https://www.journaldev.com/129/how-to-create-immutable-class-in-java">immutable</a> in Python, so this function will return a new string and the original string will remain unchanged.</p>
<pre><code>s = 'abc12321cba'

print(s.replace('a', ''))
</code></pre>
<p>Output: <code>bc12321cb</code></p>
<h3><span id="python-remove-character-from-string-using-translate">Python Remove Character from String using translate()</span></h3>
<p><a href="https://www.journaldev.com/14385/python-string">Python string</a> translate() function replace each character in the string using the given translation table. We have to specify the Unicode code point for the character and ‘None’ as a replacement to remove it from the result string. We can use <a href="https://www.journaldev.com/22767/python-ord-chr">ord() function</a> to get the Unicode code point of a character.</p><div class="code-block code-block-5" style="margin: 8px auto; text-align: center; display: block; clear: both;">
<div data-format="300x250" data-publisher="www.journaldev.com" data-type="ad" data-zone="jd_post_mid_300x250"></div></div>
<pre><code>s = 'abc12321cba'

print(s.translate({ord('a'): None}))
</code></pre>
<p>Output: <code>bc12321cb</code></p>
<p>If you want to replace multiple characters, that can be done easily using an iterator. Let’s see how to remove characters ‘a’, ‘b’ and ‘c’ from a string.</p>
<pre><code>s = 'abc12321cba'

print(s.translate({ord(i): None for i in 'abc'}))
</code></pre>
<p>Output: <code>12321</code></p>
<h2><span id="removing-spaces-from-a-string">Removing Spaces from a String</span></h2>
<pre><code>s = ' 1 2 3 4 '
print(s.replace(' ', ''))  # 1234
print(s.translate({ord(i): None for i in ' '}))  # 1234
</code></pre>
<h2><span id="python-remove-newline-from-string">Python Remove newline from String</span></h2>
<pre><code>s = 'ab\ncd\nef'
print(s.replace('\n', ''))
print(s.translate({ord('\n'): None}))
</code></pre>
<h2><span id="remove-substring-from-string">Remove substring from string</span></h2>
<p>String replace() function arguments is string. Let’s see how to remove a word from a string.</p>
<pre><code>s = 'ab12abc34ba'
print(s.replace('ab', ''))
</code></pre>
<p>Output: <code>12c34ba</code></p>
<h2><span id="remove-specified-number-of-times">Remove specified number of times</span></h2>
<p>We can also pass a third parameter in replace() function to specify the number of times replacement should be performed.</p>
<pre><code>s = 'abababab'
print(s.replace('a', 'A', 2))
</code></pre>
<p>Output: <code>AbAbabab</code></p>
